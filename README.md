# Restaurant SQL Analysis

Projeto de análise de dados utilizando MySQL.

## Objetivo
Analisar dados de pedidos e cardápio de um restaurante para responder perguntas de negócio, como:
- itens mais e menos pedidos
- categorias mais populares
- pedidos com maior gasto

## Tecnologias
- MySQL
- SQL

## Estrutura
- insert_data.sql → inserção dos dados.
- 01_menu_analysis.sql → análise do cardápio.
- 02_order_analysis.sql → análise dos pedidos.
- 03Final_analysis.sql → análise de conclusão.
